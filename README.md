# BS3206-Advanced-Software-Development

React.js Frontend - Theme: https://mui.com/store/items/minimal-dashboard-free/

Express.js Backend

MongoDB Database

## Setup

1. Copy the [.env.template](./.env.template)
1. Rename to `.env`
1. Populate the values in the `.env`
1. `npm install`

## Run

```sh
npm start
```
